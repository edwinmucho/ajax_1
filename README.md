Q. 댓글 달기 using AJAX

1. Input Tag에 댓글을 입력한다.
   1-1. Submit 버튼을 클릭한다.
   1-2. Input Tag의 값을 가져온다.
   1-3. 값이 유효한지 확인한다. (빈칸체크)
   1-4. 값이 없으면 안내메시지를 뿌린다.

2. AJAX로 처리한다. 작성자는 누군지 파악한다.
   2-1. DB에 저장한다.
3. Sever에서 처리가 완료되면 화면에 댓글을 출력한다.
